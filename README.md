# Courtroom Chaos: A Retro Pixel Decision Game

An infinite "Sort the Court" style loop set in a courtroom. Make tough decisions as a Judge, Prosecutor, or Witness and try to keep the kingdom from falling into chaos!

## 🎮 Features

- **Retro Pixel Aesthetic**: Hand-crafted UI using the "Press Start 2P" font.
- **Dynamic Scenarios**: Over 7 unique cases featuring characters like the Accused, Suspect, and Witness.
- **Cinematic Experience**: 16:9 aspect ratio with smooth cross-fade transitions between scenes.
- **Stat Management**: Balance **Trust**, **Economy**, and **Chaos**. If any hit 0 or 100, it's game over!
- **Juicy UI**: Typewriter text reveal, screen shakes, and interactive pixel buttons.

## 🚀 Tech Stack

- **React** + **Vite**
- **Tailwind CSS** (for layout)
- **Framer Motion** (for animations)
- **Typewriter-effect** (for dialogue)

## 🛠️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mahir08107/court-scene.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🖼️ Assets

The game expects image assets in `public/assets/`. Ensure you have the following files:
- `Accused Scene.png`
- `Suspect Scene.png`
- `Witness Scene.png`
